Permutation 순열
1,2,3,4나 1,2,4,3이 같음

1. index가 끝인지 확인
2. 반복문을 시작 지점부터 모두 돌면서
   3. visited 확인
      4. visited 체크
      5. 재귀 함수 (다음)
      6. visited 해제

중복으로 뽑으려면 visited를 확인하지 않으면 된다.

Combination 조합
1,2,3,4,나 1,2,4,3이 다름

1. index가 끝인지 확인
2. 반복문을 현재부터 끝까지 돌면서 (i = current)
   3. 다음 녀석 추가
   4. 재귀 함수 (다음)
   5. 다음 녀석 제거

이 때 중복으로 뽑으려면 반복문을 자신의 다음부터가 아닌 자신부터 시작하면 된다.
